# Video Background

Place your video file here with the name `bg_seniorFloors.mp4` or `bg_seniorFloors.webm`

The Hero section will automatically use this video as background.

Supported formats:
- MP4 (recommended)
- WebM (for better compression)

The video should be optimized for web:
- Resolution: 1920x1080 or higher
- Codec: H.264 (for MP4)
- Bitrate: 2-5 Mbps recommended
- Duration: Can be short and looped
